## 🔥 Main Highlight

The strongest highlight this semester was the **UTHM Inventory System**, where I moved beyond normal application development into **enterprise architecture and solution design**.

The project involved designing an integrated inventory subsystem with centralized data, SSO, repair workflow automation, QR-based asset reporting, SAP BTP concepts, and dashboard features. My focus was on system design, ERD, architecture modelling, and the repair subsystem logic.
