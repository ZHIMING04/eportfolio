## 🛠️ Coursework

### ✅ Lab 1: Daily Activities Recorder App

For Lab 1, I developed a simple **Flutter mobile application** that allows users to record and manage daily activities.

The app includes:

* Activity name input
* Time or category input
* Add activity function
* Activity list display
* Delete activity function
* Temporary in-memory storage using a `List`
* UI refresh using `setState()`

This lab helped me practice beginner-friendly Flutter concepts such as **StatefulWidget, TextField, Button, ListView, Card, and basic state management**.

## 🎯 My Assigned Module

### Taking Attendance Module

My assigned industry module was **Taking Attendance**.

This module allows lecturers to record student attendance for each class session by selecting the class, subject, and date.

Supported attendance statuses include:

| Status  | Meaning                                     |
| ------- | ------------------------------------------- |
| Present | Student attended class                      |
| Absent  | Student did not attend                      |
| MC      | Medical Leave, counted as valid attendance  |
| CK      | Cuti Kebenaran, counted as valid attendance |

The module ensures attendance data is recorded consistently and becomes the foundation for reporting, monitoring, and student performance analysis.
