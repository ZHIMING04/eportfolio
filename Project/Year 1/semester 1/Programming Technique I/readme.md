# Programming Technique I

## 📘 Course Overview

**Programming Technique I (SECJ1013)** introduced the fundamentals of structured programming and problem solving using **C++**.  
This course helped me build a strong foundation in algorithmic thinking, pseudocode, flowcharts, input/output handling, control structures, functions, arrays, pointers, and structured data types.

It was my first step in turning logical problems into working programs.

---

## 🛠 Coursework & Lab Work

Throughout the course, I completed several programming exercises and assignments involving both program design and C++ implementation.

### Key Work Completed

- Designed flowcharts from pseudocode
- Wrote pseudocode for real-world calculation problems
- Built basic C++ programs using variables, input/output, loops, and conditions
- Practiced debugging by identifying syntax and logic errors
- Used arrays and loops to process repeated data
- Created a C++ program to calculate Euclidean distance between coordinate points
- Applied functions and string handling for cleaner program structure

One of the key exercises involved calculating the distance between multiple points using the Euclidean Distance Formula, which helped me apply mathematical logic inside a C++ program.
