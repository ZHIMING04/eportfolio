# Digital Logic — Lift Electronic Controller System

## 📘 Course Overview

**Digital Logic (SECR1013)** introduced the foundation of digital electronics and how computer-based systems operate at the logic-circuit level.  
Through this course, I learned key concepts such as number systems, logic gates, Boolean algebra, combinational circuits, sequential circuits, latches, flip-flops, counters, shift registers, and programmable logic devices.

This subject helped me understand how low-level binary logic can be used to design real-world electronic systems.

---

## 🛠 Coursework: Lift Electronic Controller System

For the final project, my team designed and implemented a **Lift Electronic Controller System** using **DEEDS** simulation software.

The system was designed to control lift movement between floors using digital logic components such as:

- Logic gates
- 4-bit decoders
- 3-bit up/down counter
- 3-bit comparator
- 4-bit arithmetic comparator
- Clock disabler
- 7-segment display
- LED indicators

### Key Features

- Floor selection from level 0 to 7
- 4-digit password verification
- Emergency stop function
- Maximum weight limit detection
- LED indicators for system status
- 7-segment display for current floor output

My main contribution was the **system implementation**, where I worked on connecting the input switches, password logic, emergency stop, weight sensor, floor selection, and counter control into one functioning lift controller system.
