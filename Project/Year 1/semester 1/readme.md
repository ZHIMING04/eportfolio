# Year 1 Semester 1 — Computing Foundations

## 📘 Semester Overview

Year 1 Semester 1 marked the beginning of my Computer Science journey at Universiti Teknologi Malaysia.  
This semester focused on building strong fundamentals in programming, mathematical logic, digital systems, and information technology.

Through these subjects, I developed my early technical foundation in **C++ programming, structured problem-solving, discrete mathematics, digital logic circuit design, information systems, and design thinking**.

This semester helped me understand how computing works from multiple layers:  
from writing code, to applying logic, to understanding hardware circuits, to designing technology solutions for real users.

---

## 📚 Subjects Taken

### 💻 Programming Technique I

This subject introduced me to structured programming using **C++**.  
I learned how to solve problems step by step using pseudocode, flowcharts, variables, input/output, control structures, loops, functions, arrays, pointers, and structured data types.

#### Coursework Highlights

- Designed flowcharts from pseudocode
- Wrote pseudocode for real-world calculation problems
- Built basic C++ programs
- Practiced debugging syntax and logic errors
- Used functions, loops, arrays, and mathematical operations
- Created a program to calculate Euclidean distance between coordinate points

#### Skills Gained

- C++ programming
- Structured problem-solving
- Pseudocode and flowchart design
- Debugging
- Logical thinking
- Mathematical computation in programming

### 🔢 Discrete Structure

This subject strengthened my mathematical and logical thinking for Computer Science.  
I learned core topics such as set theory, logic, proof techniques, relations, functions, recurrence relations, counting methods, probability, graph theory, trees, and finite automata.

#### Skills Gained

- Logical reasoning
- Mathematical problem-solving
- Set theory and relations
- Counting and probability
- Graph and tree concepts
- Critical thinking for computing problems

---

### ⚙️ Digital Logic

This subject introduced the foundation of digital electronics and computer hardware logic.  
I learned how binary logic, logic gates, Boolean algebra, combinational circuits, sequential circuits, flip-flops, counters, and shift registers are used to build digital systems.

#### Coursework Highlight: Lift Electronic Controller System

For the final project, my team designed a **Lift Electronic Controller System** using DEEDS simulation software.

The system included:

- Floor selection from level 0 to 7
- 4-digit password verification
- Emergency stop function
- Maximum weight limit detection
- LED indicators
- 7-segment display for current floor output
- Counter and comparator logic

#### Skills Gained

- Digital circuit design
- Boolean logic
- Combinational circuit design
- Sequential circuit design
- Counter and comparator logic
- DEEDS simulation
- System-level thinking

### 🌐 Technology & Information System

This subject introduced me to information systems, computer hardware, software, networks, databases, cloud computing, privacy, security, ethics, and IT career pathways.

It also exposed me to industry talks, industry visits, PC assembly, GitHub e-portfolio development, and design thinking.

#### Coursework Highlight: UTM EDU

For the Design Thinking project, my team proposed **UTM EDU**, a student-focused educational software prototype designed to improve campus life.

The prototype included:

- Event information and registration
- Sports facility booking
- Bus schedule and transport notification
- Cafe and restaurant information
- Important date reminders
- Feedback and survey feature

#### Skills Gained

- Problem identification

## 🧠 Semester Reflection

Year 1 Semester 1 gave me the foundation I needed to start thinking like a Computer Science student.

Programming Technique I taught me how to break problems into logical steps and implement them using C++.  
Discrete Structure trained my mathematical reasoning and helped me understand the logic behind computing problems.  
Digital Logic showed me how computers operate at the hardware and binary logic level.  
Technology & Information System helped me understand how technology is applied in real organizations and how user-centered solutions can solve real problems.

The biggest lesson from this semester was that computing is not only about coding.  
It involves logic, systems, hardware, users, and problem-solving.

This semester built the foundation for my future learning in software development, data engineering, cloud computing, and system design.
