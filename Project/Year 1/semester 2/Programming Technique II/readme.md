## 📘 Course Overview

**Programming Technique II (SECJ1023)** introduced the concept of **Object-Oriented Programming (OOP)** using the C++ programming language.

This course built on the foundation from Programming Technique I and shifted my programming approach from procedural thinking to object-oriented design.

Through this subject, I learned how to design and develop programs using core OOP principles such as **encapsulation, abstraction, association, aggregation, composition, inheritance, and polymorphism**.

---

## 🛠️ Coursework

Throughout the course, I completed exercises and project deliverables that focused on applying OOP concepts in C++.

### Key Topics Covered

| Topic                                  | Description                                                                      |
| -------------------------------------- | -------------------------------------------------------------------------------- |
| Classes & Objects                      | Defining classes, creating objects, private members, and class implementation    |
| Constructors & Destructors             | Initializing and destroying objects properly                                     |
| Object Manipulation                    | Friend classes, pointers to objects, arrays of objects, and operator overloading |
| String Manipulation                    | Using C++ string class and string member functions                               |
| Association, Aggregation & Composition | Modelling relationships between objects                                          |
| Inheritance                            | Creating base and derived classes                                                |
| Polymorphism                           | Using virtual functions, abstract classes, and dynamic behaviour                 |
| Advanced Features                      | Introduction to exceptions, templates, containers, and iterators                 |

## 🚀 Mini Project

The course included a group mini project where we applied object-oriented programming concepts to develop a medium-complexity C++ program.

The project required:

* Problem analysis
* OOP-based design
* UML class diagram planning
* Class implementation
* Team collaboration
* GitHub-based teamwork tracking
* Final project implementation

This project helped me understand how OOP is used to organize larger programs into reusable and maintainable components.

## 🧠 Reflection

Programming Technique II helped me move beyond basic programming and understand how larger software systems are structured.

The biggest shift was learning to think in terms of **objects, responsibilities, and relationships** instead of only writing step-by-step procedures. Concepts like inheritance, composition, and polymorphism made me realize how software can be designed to be more reusable, organized, and easier to maintain.

This course strengthened my foundation in software development and prepared me for more advanced subjects involving data structures, system development, and backend programming.
