## 🚀 Semester Overview

Year 1 Semester 2 expanded my foundation from basic computing into **software structure, computer architecture, UX design, and system analysis**.

This semester taught me how to think beyond code by understanding how systems are designed, how users interact with applications, and how hardware affects software performance.
