## 📘 Course Overview

**Human Computer Interaction (SECV2113)** introduced the principles of designing interactive systems that are useful, usable, accessible, and meaningful to users.

Through this course, I learned about **user experience design, human behaviour, interaction design, requirements gathering, prototyping, usability testing, evaluation methods, and iterative improvement**.

This subject helped me understand that good technology is not only about functionality, but also about how users think, feel, behave, and interact with a system.

---

## 🛠️ Coursework

### 🌱 Tree-Huggers United

For the course project, my team created **Tree-Huggers United**, a UX-focused prototype designed to promote environmental awareness and encourage users to participate in eco-friendly actions.

The project explored how digital platforms can support sustainability by helping users understand environmental issues, engage with green activities, and connect with relevant information in a more accessible way.

## 🎯 Project Focus

The project was developed around several key HCI and UX activities:

| Stage                  | Description                                                                |
| ---------------------- | -------------------------------------------------------------------------- |
| Problem Identification | Studied user problems related to environmental awareness and participation |
| User Research          | Identified potential users and their needs                                 |
| Ideation               | Generated design ideas to improve environmental engagement                 |
| Prototyping            | Created interface sketches and Figma prototype                             |
| Usability Thinking     | Considered how users would navigate and interact with the system           |
| Evaluation             | Reflected on design clarity, usefulness, and improvement areas             |

---

## 👥 Target Users

The prototype was designed with several user groups in mind:

* Students
* Environmental volunteers
* General public users interested in sustainability
* Organizations or communities promoting green initiatives

## 🧠 Reflection

This project helped me understand that designing a system is not just about making it look good. A good interface must solve a real user problem, communicate clearly, and guide users naturally toward their goals.

Through **Tree-Huggers United**, I learned how to apply HCI concepts such as user needs, usability, accessibility, prototyping, and evaluation into a real design project. The process also trained me to think from the user’s perspective instead of only focusing on features.

Overall, this course improved my ability to design more user-friendly systems and strengthened my awareness of how UX decisions can influence user engagement, especially in projects with social and environmental impact.
