## 🔥 Main Highlight: KADA eServe

The biggest project this semester was **KADA eServe**, a system developed by **Team TechAway** for Koperasi Kakitangan KADA.

Across Database, Software Engineering, and System Development Technology, the project evolved from:

**analysis → database design → software proposal → system architecture → Laravel implementation → testing**

The system focused on digitizing manual workflows such as:

* Member registration
* Loan application
* Savings management
* Transaction records
* Individual reports
* Annual report upload
* Admin approval workflow

## I contributed as **System Development Lead / SDT Leader**, focusing on system planning, Agile development flow, Laravel MVC implementation, and system integration.

## 🧠 Reflection

Year 2 Semester 1 helped me connect theory with real implementation.

I learned that building a useful system requires more than coding. It needs strong database design, clear requirements, reliable networking knowledge, structured software planning, and careful testing.

This semester gave me a stronger foundation in **backend development, database systems, networking, and full-stack system development**, which directly supports my path toward **Data Engineering, Cloud, and scalable system building**.
