## 📘 Course Overview

**System Development Technology (SDT)** introduced the practical development of database-driven web applications.

Through this course, I learned how to build dynamic web systems using **HTML, CSS, JavaScript, PHP, MySQL, MVC architecture, Laravel, authentication, CRUD operations, validation, file handling, reporting, and deployment**.

This subject helped me move from system planning into real implementation, where requirements were translated into a working web application.

---

## 🛠️ Coursework

### 🚀 KADA eServe System

For the main project, my team **TechAway** developed **KADA eServe**, a web-based system for **Koperasi Kakitangan KADA**.

The system was designed to replace manual paper-based workflows with a digital platform for:

* Member registration
* Loan application
* Loan status tracking
* Member profile management
* Savings and transaction management
* Annual report upload
* Individual report viewing
* Admin approval workflow

## 🧠 Reflection

This course was one of the most practical subjects in my Year 2 journey.

Through **KADA eServe**, I learned how to turn system requirements, diagrams, and database designs into a working web application. The project gave me hands-on experience with Laravel, MySQL, MVC architecture, authentication, form validation, reporting, and deployment preparation.

As the SDT Leader, I also learned that system development is not only about writing code. It requires coordination, debugging, documentation, testing, communication, and continuous improvement based on feedback.

Overall, this subject strengthened my full-stack development foundation and gave me valuable experience in building a real-world database-driven system.
