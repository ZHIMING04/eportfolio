## 📘 Course Overview

**Data Structures and Algorithms (SECJ2013)** introduced the core concepts of organizing, processing, and analyzing data efficiently.

This course strengthened my understanding of **abstract data types, recursion, algorithm efficiency, Big O notation, sorting, searching, linked lists, stacks, queues, trees, and binary search trees**.

It helped me move from simply writing programs to thinking about how efficient and scalable a solution is.

---

## 🛠️ Coursework

### Lab 1 — Student Result Processing

In this lab, I developed a C++ program using a `Student` class to process student marks from a file.

The program included:

* Reading student data from `Marks.txt`
* Creating student objects dynamically
* Calculating total marks
* Assigning grades
* Printing results to console
* Saving results into `Results.txt`
* Practicing object-oriented programming with file handling

### Lab 2 — Recursion Practice

In this lab, I practiced recursive problem solving using C++.

The program included:

* Recursive star pattern printing
* Recursive number pattern generation
* Recursive calculation of total odd numbers in an array

## 🧠 Reflection

This course helped me understand that good programming is not only about getting the correct output, but also about choosing the right structure and algorithm for the problem.

Through the labs, I practiced writing cleaner C++ programs using classes, files, arrays, and recursion. The course also trained me to think about efficiency, especially when comparing different algorithms using Big O notation.

Overall, Data Structures and Algorithms gave me a stronger foundation for software development, backend logic, and future data engineering work.
