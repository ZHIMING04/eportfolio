## 📘 Course Overview

**Object Oriented Programming (OOP)** introduced the concepts and techniques of building software using the Java programming language.

Through this course, I learned how to design Java applications using **classes, objects, encapsulation, constructors, arrays, collections, inheritance, polymorphism, interfaces, exception handling, and selected Java APIs**.

This subject helped me move from writing simple programs into designing reusable, modular, and maintainable object-oriented applications.

---

## 🛠️ Coursework & Lab Work

### 🧩 Exercise: OOP Principles

In the early exercise, I explored the basic principles of object orientation, including:

* Divide-and-conquer
* Encapsulation
* Modularity
* Public interface
* Information hiding
* Generality
* Extensibility
* Abstraction

I also reflected on OOP benefits such as **reusability, maintainability, easier debugging, and better software organization**.

---

### 💻 Lab 2: Classes, Objects & Encapsulation

This lab focused on Java class design and object manipulation.

Key practices included:

* Creating Java classes
* Using private attributes
* Writing getter and setter methods
* Creating constructors
* Applying method overloading
* Using static variables and methods
* Practicing encapsulation through employee, car, book, student, message, and user examples

---

### 📊 Lab 3: Arrays, ArrayList & Object Arrays

This lab focused on handling data collections in Java.

Key practices included:

* One-dimensional arrays
* Two-dimensional arrays
* ArrayList usage
* Passing arrays into methods
* Returning and processing values
* Finding the highest score
* Calculating averages
* Summing student marks
* Creating and processing arrays of `Student` objects

## 🧠 Reflection

This course strengthened my understanding of how software can be designed using objects instead of only procedures.

Through the labs, I practiced building Java classes with clear attributes, methods, constructors, and relationships. I also became more comfortable with managing collections of data using arrays, ArrayList, and object arrays.

The biggest takeaway was learning that OOP is not just a programming style. It is a way to organize logic, reduce duplication, improve maintainability, and prepare programs for future extension.

Overall, Object Oriented Programming improved my Java foundation and helped me think more clearly about software structure, reusability, and system design.
