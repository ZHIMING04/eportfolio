## 🚀 Project: CO₂ Emissions From Cars

For the main project, my team analyzed **car CO₂ emissions** using two datasets:

* `FuelConsumption.csv`
* `cars_2025.csv`

The research question focused on how different car makers and models compare in terms of carbon emissions, and whether they can be classified based on emission-related attributes.

### Project Phases

| Phase   | Description                                              |
| ------- | -------------------------------------------------------- |
| Phase 1 | Data cleaning, preparation, merging, and wrangling       |
| Phase 2 | Analytics, visualization, machine learning, and insights |

---

## 🔍 Key Analysis Performed

* Cleaned and standardized car maker and model names
* Removed duplicates and missing values
* Merged car registration data with fuel consumption data
* Analyzed average CO₂ emissions by car maker
* Compared emissions by car model
* Grouped emissions by engine size
* Compared emissions by vehicle class
* Analyzed emissions by transmission type
* Built a Random Forest model to classify car makers based on emission-related features

## 🧠 Reflection

This course helped me understand how raw data can be transformed into useful insights.

Through the CO₂ emissions project, I learned that data analysis is not only about writing Python code. It involves cleaning messy datasets, choosing meaningful features, asking the right questions, visualizing patterns, and explaining the findings clearly.

The project also gave me early exposure to machine learning, where I used Random Forest to classify car makers based on features such as engine size, fuel consumption, cylinders, transmission, and CO₂ emissions.

Overall, Data Analytics Programming strengthened my foundation in Python, Pandas, data preparation, analytics, and machine learning — all important skills for my data engineering journey.
