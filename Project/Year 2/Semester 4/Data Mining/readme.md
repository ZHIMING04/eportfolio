## 📘 Course Overview

**Data Mining** introduced the process of discovering meaningful patterns and insights from data.

Through this course, I learned about **Knowledge Discovery in Databases (KDD), CRISP-DM, data preprocessing, classification, clustering, association rule mining, and model evaluation**.

This subject helped me understand how machine learning techniques can be used to solve real-world analytical problems and generate business insights from raw data.

---

## 🛠️ Coursework

### 🎧 Project Dataset: Spotify Tracks Dataset

For the main projects, my team analyzed the **Spotify Tracks Dataset** from Kaggle, which contains Spotify songs, 114 genres, and multiple audio features.

We used Python Notebook instead of drag-and-drop data mining tools because it gave us better control over preprocessing, modelling, evaluation, and visualization.

---

## 🚀 Project 1: Classification

Project 1 focused on **supervised learning**.

We applied three classification algorithms:

* Support Vector Machine
* Decision Tree
* Random Forest

### Business Insights Explored

| Insight                 | Goal                                                     |
| ----------------------- | -------------------------------------------------------- |
| Hit Song Prediction     | Classify tracks as hit or non-hit based on popularity    |
| Danceability Prediction | Classify songs into low, medium, or high danceability    |
| Instrumentation Level   | Classify songs as vocal-heavy, balanced, or instrumental |

Across the classification tasks, **Random Forest** achieved the strongest overall performance compared to SVM and Decision Tree.

---

## 🚀 Project 2: Clustering

Project 2 focused on **unsupervised learning**.

We applied two clustering algorithms:

* K-Means
* Gaussian Mixture Model

### Business Insights Explored

| Insight                 | Goal                                                                        |
| ----------------------- | --------------------------------------------------------------------------- |
| Mood Clustering         | Group songs into moods such as calm, chill, party, melancholic, and intense |
| Karaoke Difficulty      | Cluster songs by karaoke difficulty level                                   |
| Artist Style Clustering | Group artists based on their music characteristics                          |

Across the clustering tasks, **K-Means** consistently performed better than GMM by producing more compact and well-separated clusters.

## 🧠 Reflection

This course helped me understand that data mining is not just about applying algorithms. The real challenge is preparing the data, selecting meaningful features, choosing suitable models, and interpreting results correctly.

Through the Spotify projects, I experienced real machine learning challenges such as **overfitting, feature engineering, imbalanced classes, clustering evaluation, and memory limitations** when working with large datasets.

Overall, Data Mining strengthened my foundation in machine learning, model evaluation, and insight generation. It also gave me a clearer view of how raw data can be transformed into useful knowledge for decision-making.
